我的装备 / Uses

工欲善其事，必先利其器。这是我日常开发和工作中使用的软硬件清单。

硬件设备

主力笔记本

MacBook Pro 14寸 M1 Pro。性能强劲，续航无敌，完美胜任数据分析和前端开发。

键盘

Keychron K6。小巧便携，支持蓝牙多设备切换，红轴手感舒适。

开发工具

编辑器：VS Code

目前最爱的编辑器。配合 Vim 插件和 Remote SSH，开发效率极高。

终端：iTerm2 + Oh My Zsh

程序员必备。配置了 Powerlevel10k 主题，美观且功能强大。

数据库：DataGrip

JetBrains 出品，必属精品。对 MySQL 和 PostgreSQL 的支持非常完善。

效率工具

Notion

我的第二大脑。用于知识管理、项目规划以及日常笔记。

Raycast

Spotlight 的替代品。脚本功能让我能快速执行一些自动化任务。
