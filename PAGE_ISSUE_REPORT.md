## 七、修复建议

### 7.1 高优先级

1. **简历PDF**: 添加真实PDF到public目录
2. **分类筛选**: 实现工具页和项目页的分类筛选JS逻辑
3. **表单功能**: 接入Formspree/EmailJS或显示联系邮箱
4. **移动导航**: 检查并修复汉堡菜单点击事件

### 7.2 中优先级

1. **API Key**: 改用password类型输入框
2. **Giscus**: 确保通过postMessage正确切换语言
3. **项目链接**: 使用真实GitHub仓库URL

### 7.3 低优先级

1. **性能**: 考虑代码分割和动态导入
2. **安全**: API Key考虑使用加密存储
3. **体验**: Loader添加跳过按钮
